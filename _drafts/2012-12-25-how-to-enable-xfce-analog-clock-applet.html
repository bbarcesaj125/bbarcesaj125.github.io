You like the Xfce analog clock applet and want to enable it in Unity? If that is the case with you, then follow the instructions below.

In this tutorial, I will walk you through the process of installing and running the Xfce panel widget in Ubuntu.

First, you need to install the required packages:

sudo apt-get install xfce4-panel wmctrl

Now, you need to configure the panel:

#Launch the panel from the terminal:


xfce4-panel

#Create an empty panel:


When you run the xfce4-panel for the first time, it will ask you to create a new panel:


After you have created an empty panel, you need to place it in the App Indicator area:


#Edit the panel options:

Set the panel height to 24 px


Next, you need to apply a solid color background to the panel and set the alpha channel to zero (full transparency):


#Choose your windget/s:

Now, you can easily add items to your panel just like you would do it in a native Xfce environment:


When add an item to the panel, you will notice that it disappear when you invoke the Unity dash. To solve this problem, you can run this command in a terminal:

wmctrl -r xfce4-panel -b add,above

#Change the clock color:

Put the following lines in your ~/.gtkrc-2.0 file:

style "panel"
{
    fg[NORMAL] = "#CED8EB"
}

widget_class "*Panel*"      style "panel"
widget "*Panel*"            style "panel"
class "*Panel*"             style "panel"

If the file doesn't exist then create one.

#Make Permanent Changes:

In order to start the Xfce panel automatically, you have to create a script that will execute the xfce4-panel each time you start your system.

Create a .sh file with your favorite text editor, then paste the following text into it:

#!/bin/bash
xfce4-panel &
sleep 3;
wmctrl -r xfce4-panel -b add,above
&

Make the resulting file executable:

sudo chmod +x myXfcePanelStartupScript.sh

We are almost done, all you have to do now is adding the script to your startup applications.
