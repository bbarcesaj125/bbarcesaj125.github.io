Uget is a cross-platform download manager written in GTK+. Uget provides an easy to use interface and a lot of features such as:
  • Resume download support
  • Create download categories
  • Download/Upload speed control
  • Command Line interface
  • Download schedule support
  • Batch downloads
  • Clipboard monitoring (automatically insert the copied URL)
  • App Indicator support 

     

    Firefox Integration:  

     

     

    Uget integrates easily with Firefox through the FlashGot addon which you can install here: https://addons.mozilla.org/en-US/firefox/addon/flashgot/
    To download a file with Uget, just select "Uget" from the dropdown menu and the download will start automatically.

    Uget comes with NotifyOSD support:

     

    Uget is  a cross-platform application. To install on Ubuntu, add the following PPA to your Software Sources:

    sudo apt-add-repository ppa:plushuang-tw/uget-stable 
    sudo apt-get update 
    sudo apt-get install uget

    For more information about the program, visit the official Uget site:
    https://uget.visuex.com/
