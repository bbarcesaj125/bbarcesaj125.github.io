Mundus is a small GUI utility which lets you keep your $HOME folder clean by deleting the configuration files of uninstalled applications. It maintains a database of popular applications allowing you to painlessly remove unused configuration files/folders.

Among Mundus features are:

  • Backup support
  • Clean Interface
  • Preview configuration files in your file manager
  • Faenza Icons (For displaying installed applications)

Installation:

To install Mundus on Ubuntu, simply issue the following commands into a terminal window:

sudo add-apt-repository ppa:sebikul/mundus-stable
sudo apt-get update
sudo apt-get install mundus

Warning:

To prevent data loss, it is highly recommended that you backup your configuration files before cleaning (i.e. using Mundus).
