TreeLine is a multi-purpose note taking application that you can use to keep a structured list of notes. It is a simple yet feature-rich application that can be used to store and manage any type of information (e.g. notes).

Examples Of Its Usage Include:

  • Todo list application
  • Collection manager (e.g. books, movies ...)
  • Personal contacts organizer
  • Brainstorming
  • Taking notes
  • Learning a new language
