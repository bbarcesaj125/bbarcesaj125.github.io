Variety is python application that was created by Peter Levi as part of the previous Ubuntu App Showdown. It allows the user to change the wallpaper dynamically by using either a local or remote wallpaper source.

Variety comes bundled with a lot of features such as:
  • Changing wallpaper using predefined intervals
  • Downloading wallpapers from various online sources
  • Sharing Downloaded wallpapers on Facebook
  • Applying cool effects to the wallpapers

Variety's Noticeable Features:

#App Indicator


Variety includes an app indicator that allows you to rapidly edit some options such as:
  • Controlling the playback :

  • Viewing the source
  • Sending the wallpaper to Facebook

#Online Sources


Variety can also fetch wallpapers from various online sources:


The list of supported online sources includes:

  • Flickr
  • Desktoppr . Co
  • NASA's Astronomy Picture Of The Day
  • Live Earth Wallpaper from Opentopia . Com
  • Wallbase . Cc
  • Media RSS
  • Wallpapers . Net
  • Etc

#Fancy Effects


Among the cool effects you can choose from are:
  • Grayscale
  • Heavy Blur
  • Oil Painting
  • Charcoal Painting
  • Pointilism
  • Pixellate
  • Clock (Show a digital clock on the bottom right corner of the screen) :

To install Variety on Ubuntu, add the following PPA to your software sources:

sudo add-apt-repository ppa:peterlevi/ppa
sudo apt-get update
sudo apt-get install variety
