DeaDBeef is a simple music player for Ubuntu/Linux that supports both GTK+ 2 and GTK+ 3. It is a very lightweight application which can play almost any audio format.

DeaDBeef includes the following functionalities:

  • Super graphical equalizer (18 bands) 
  • DSP plugins
  • Minimal group of dependencies (No GNOME or KDE dependencies)
  • Tabbed interface
  • Small memory usage
  • Internal tag editor
  • Built-in Encoder (i.e. Format converter)
  • Last FM support
  • OSD notifications
  • Global hotkeys 
  • Streaming radio support
