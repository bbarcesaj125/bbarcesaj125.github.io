Speed Dreams is an open source racing game. The latest version was released on April 8.
Speed Dreams was forked from Torcs which is a 3D Car racing simulation game.
Both Torcs and the forked version support a wide range of input devices such as keyboards, mouses, joysticks and pedals.


Speed Dreams features several racing modes which can be associated with various types of championships. The latest version of Speed Dreams introduced a "career" mode which gives users the possibility to play against random opponents with different car classes.

Here's a list of Speed Dreams features :
  • Wide range of tracks
  • Customizable weather conditions
  • Different camera angles
  • Etc
Speed Dreams is a cross-platform game, so it is available on Windows, Linux and Mac (although the Mac port isn't finished yet). To install it on Ubuntu, add the following ppa to your software sources:

sudo apt-add-repository ppa:speed-dreams/ppa 
sudo apt-get update 
sudo apt-get install speed-dreams
