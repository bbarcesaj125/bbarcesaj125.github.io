SubDownloader is a lightweight program to automatically download/upload subtitles for video files. It provides a clean interface which lets you find matching subtitles for your movies in just a few steps.

Among SubDownloader useful features are:
  • Subtitles language auto-detection
  • Ability to play videos directly using an external player 
  • Fast indexing algorithm
  • Subtitles upload support


SubDownloader is a cross-platform application. To install it on Ubuntu, you need to add the following ppa to your Software Sources:

sudo apt-add-repository ppa:subdownloader-developers/ppa 
sudo apt-get update 
sudo apt-get install subdownloader
