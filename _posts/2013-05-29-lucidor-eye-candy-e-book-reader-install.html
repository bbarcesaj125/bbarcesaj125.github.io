Lucidor is a simple application for reading and managing E-books. The application comes with a slick interface which allows you to easily read your EPUB files and manage your OPDS collections.

Some of Lucidor features include:

  • Collection manager (i.e. local bookcases)
  • Online catalog browser
  • Tabbed interface
  • Built-in E-book converter (e.g. convert web feeds/pages to E-books)
  • Theme support
Lucidor also comes with some unique features such as the ability to use Firefox themes (from Firefox Add-ons). However, it is worth mentioning that not all themes work properly, so if want to get a working theme, you can get it from Lucidor's official website HERE.
