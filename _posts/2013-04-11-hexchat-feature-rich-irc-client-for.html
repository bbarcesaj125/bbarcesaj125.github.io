Hexchat is a GTK+ IRC client for Ubuntu/Linux and Windows. It is an actively developed project which was derived from XChat back in 2010. Furthermore, it doesn't only include its ancestor features (i.e. XChat) but adds new ones as well.

Some of HexChat's features include:

  • Customizable interface 
  • Built-in server list (Freenode, GalaxyNet, GameSurge ...)
  • Extensible (Plugins & scripts)
  • DCC support
  • Tabbed interface
  • Auto-connect support

To install HexChat on Ubuntu, simply add the following PPA to your Software Sources:

sudo add-apt-repository ppa:gwendal-lebihan-dev/hexchat-stable
sudo apt-get update
sudo apt-get install hexchat
