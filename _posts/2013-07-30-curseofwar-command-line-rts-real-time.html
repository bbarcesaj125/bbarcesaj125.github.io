CurseOfWar is an action RTS game which incorporates a fast-paced gaming style that allows you to control your territory by focusing on high-level strategic planning. This type of high planning includes some typical war strategies such as creating infrastructure, gathering resources and controlling armies.


Although the game requires some strategic skills, it has very few rules. You can explore these rules by following this link: CurseOfWar's rules.

Installation Instructions:

To install CoW on Ubuntu, simply add the following PPA to your software sources:

sudo add-apt-repository ppa:curseofwar/curseofwar-stable
sudo apt-get update
sudo apt-get install curseofwar

After you've successfully installed the game, you can start a lightweight session (useful for beginners) by running the following command:

curseofwar -i4 -q1 -dee

For more tips about the game, please visit its home page @Github
