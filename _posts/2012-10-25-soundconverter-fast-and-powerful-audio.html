SoundConverter is a simple yet powerful sound converter for Ubuntu. It supports a wide range of media formats such as Ogg Vorbis, Mp3, AAC, FLAC, WAV, AVI, MOV, MPEG, M4A, AC3 ...etc
As for the output formats, SoundConverter can convert any of the aforementioned formats to Mp3, Opus, Ogg Vorbis, FLAC, WAV and AAC.

SoundConverter is a very fast converter since it uses a multi-threaded approach to make the conversion faster. It offers advanced features such as:
  • Automated renaming
  • Audio resampling
  • Ability to extract audio from video files






SoundConverter is available in the GetDeb repository. To add this repository and install SoundConverter, run the following commands:
(Update Jan 22, 2013: This installation method is no longer recommended because the GetDeb repository has been shut down. Use the alternative method described below.)

echo "deb http://archive.getdeb.net/ubuntu quantal-getdeb apps" | sudo tee -a /etc/apt/sources.list
wget -q -O- http://archive.getdeb.net/getdeb-archive.key | sudo apt-key add -
sudo apt-get update
sudo apt-get install soundconverter

Alternatively, you can safely install SoundConverter from the official repositories by using the following command:


sudo apt-get install soundconverter
