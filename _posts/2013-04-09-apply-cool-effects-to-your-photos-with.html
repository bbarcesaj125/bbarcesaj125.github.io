PhotoFiltreLX is a clone of the popular image editing application PhotoFiltre for GNU/Linux. It allows you to easily edit your photos using a wide range of filters such as brightness, contrast, puzzle effect ...


Features:

  • Advanced selections
  • Image editing tools (Blur, smudge, brush, bucket fill ...)
  • Filters (Artistic, relief, noise, sharpen ...)
  • Batch editing

Installation:

To install PhotoPhiltreLX on Ubuntu, simply download the .deb file (HERE), then either use a GUI tool such as GDebi or run the following command in a terminal:

sudo dpkg -i <FILENAME.deb>

IMPORTANT NOTE: PhotoFiltreLX is in an early development stage (i.e. Alpha). Therefore, some features may not work properly. It is highly recommended that you don't use it in a production machine.
