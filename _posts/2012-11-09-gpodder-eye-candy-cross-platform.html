gPodder is a lightweight podcast manager that allows you to painlessly access free audio and video content through a simple and clean interface.

gPodder comes with a built-in content discovery tool that lets you grab/download podcasts from various sources such as:
  • Linux Outlaws
  • Floss Weekly
  • BSDTalk
  • TEDTalks
  • Crunchtime on SoundCloud
  • TuxRadar Linux Podcast
To use this feature, simply press Ctrl+F, select some sources, and then click on add:


gPodder also offers a synchronization feature which you can use to keep track of your subscriptions across multiple platforms (full gpodder.net integration).

Among its other features, gPodder supports these sources:
  • RSS and Atom
  • Vimeo, SoundCloud, Youtube & XSPF feeds
gPodder is a cross-platform application which supports Linux, Windows, Mac OS X and mobile OSs such as Maemo and MeeGo.

Ubuntu users can easily install gPodder using the following PPA:

sudo apt-add-repository ppa:thp/gpodder
sudo apt-get update
sudo apt-get install gpodder
