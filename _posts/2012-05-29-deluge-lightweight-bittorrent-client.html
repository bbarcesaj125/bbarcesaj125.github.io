Deluge is a cross-platform torrent client which supports protocol encryption. The encryption feature is used to prevent your ISP from shaping your traffic.

 

Deluge comes with a lot of functionalities such as:
  • Plugin System
  • Web interface
  • Proxy support 
  • Private Torrents
  • Password protection  
The rich plugin collection and  the clean interface make Deluge a perfect choice for both average and advanced users.

Deluge is available on Linux, Windows and Mac. To install it on Ubuntu, run the following commands in a Terminal:

sudo apt-add-repository ppa:deluge-team/ppa 
sudo apt-get update 
sudo apt-get install deluge
