EmelFm2 is a double panel file manager for Linux and BSD systems. It has a GTK+ powered interface  that allows to access all the features you might expect from a file manager such as:

  • Opening the terminal in the current directory
  • Invoking commands directly from the built-in command-line
  • Showing hidden files
  • Creating symbolic links
  • Calculating disk usage of selected items
  • Etc

To install EmelFM2 on Ubuntu, simply add the following PPA to your software sources:

sudo add-apt-repository ppa:emelfm2/release
sudo apt-get update
sudo apt-get install emelfm2
