Nanoshot is a feature-rich screenshot tool for Linux. It is an easy to use program which comes with an application indicator and a lot of functionalities such as:

  • Take screenshots of video
  • Upload screenshots to popular image hosting sites (Flickr, ImageShack ...)
  • Take screenshots of web pages
  • Capture delay support



To install Nanoshot on Ubuntu,  add the following ppa to your software sources:

sudo apt-add-repository ppa:nanoshot/ppa 
sudo apt-get update 
sudo apt-get install nanoshot
