TileMill is a beautiful map designer developed by MapBox which is a company that provides tools and services for creating and hosting maps.
TileMill uses the Mapnik rendering engine. Mapnik is an open source tool designed for developing interactive maps.

Among TileMill features are:
  • Carto support (CSS-like map styling language)
  • Embedded graphs and images
  • Create clickable  pop ups
  • TrueType, OpenType and PostScript fonts support
Ubuntu users can install this application using this PPA:

sudo apt-add-repository ppa:developmentseed/mapbox 
sudo apt-get update 
sudo apt-get install tilemill
