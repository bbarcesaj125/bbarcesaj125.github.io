Vym is an open source mind mapping application. A mind map is a technique which helps you organize your ideas by arranging them around a central key word. It is a technique designed to improve productivity and creativity.
Vym lets you easily create mind maps with an easy to use interface which provides a lot of features such as:
  • Insert notes, images and flags
  • Record parts of the map
  • Export maps to HTML 

    Vym is available on Linux and Windows. To install it on Ubuntu, download the .deb file (here) to your home folder then run the following command:

    sudo dpkg -i FILENAME.deb
