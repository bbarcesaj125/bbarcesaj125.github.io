GCstar is a an open source application centered around managing collections of movies, books, comics and such. It can automatically fetch detailed informations about each item using a large database of online sources.

GCstar is able to manage the following categories/collections:


  • Board games
  • Books
  • Computer software
  • Comics
  • Movies
  • Music
  • Mini vehicles
  • Periodicals
  • Smart card
  • Stamp
  • TV shows
  • Video Games
  • Numismatic

Ubuntu users can easily instal GCstar using the following PPA:

sudo add-apt-repository ppa:gcstar/ppa
sudo apt-get update
sudo apt-get install gcstar
