The STPPC (Simon Tatham's Portable Puzzle Collection) is a huge collection of puzzle games that run natively on Ubuntu/Linux using GTK+.

The Tatham collection includes the following one-player puzzle games:

Topological deduction games:

  • Bridges
  • Loopy
  • Map
  • Net (Based on FreeNet)
  • Slant

Tile manipulation games:

  • Cube
  • Fifteen
  • Flip
  • Netslide
  • Sixteen
  • Twiddle

Miscellaneous Puzzle Games:

  • Untangle (Based on Planarity)
  • Unruly
  • Unequal (Based on Futoshiki)
  • Undead
  • Towers (Based on Skyscrapers)
  • Tents
  • Solo (Based on Sudoku)
  • Singles (Based on Hitori)
  • Signpost (Based on Pfeilpfad)
  • Same Game
  • Rectangles (Based on Divide by Squares)
  • Range (Based on Kurodoko)
  • Pegs (An implementation of several versions of peg solitaire)
  • Pearl (Based on Masyu)
  • Pattern (An implementation of nonograms)
  • Mines (Based on Minesweeper)
  • Magnets
  • Light Up
  • Keen (Based on KenKen)
  • Inertia
  • Guess (Based on Mastermind)
  • Galaxies (Based on Tentai Show)
  • Filling (Based on Fillomino)
  • Dominosa
  • Black Box

To Install this collection on Ubuntu, run the following command in a terminal:

sudo apt-get install sgt-puzzles

NOTE: The version available in the official repositories doesn't include these two games:

  • Untruly
  • Undead
