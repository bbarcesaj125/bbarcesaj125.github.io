LiveStreamer is a command-line application that allows you to play content from live stream services such as Justin.tv. It is a flash-free application which means that all you need to play streams is a media player (e.g. VLC).

LiveStreamer supports the following online services:

  • Livestream
  • Dailymotion
  • Twitch/Justin.tv
  • YouTube Live
  • Ustream
A full list of supported service can be found here: Supported Services.

Installation:

The easiest method to install LiveStreamer is by using python-pip: 

sudo apt-get install python-pip
sudo pip install livestreamer

Example:

Grab live stream from this URL: justin.tv/somechannel:

livestreamer justin.tv/somechannel best
