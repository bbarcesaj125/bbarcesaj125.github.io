Claws-Mail is a simple email client for Ubuntu. It comes with a clean GTK+ interface that was specifically designed to facilitate the transition from other email clients to Claws-Mail.

Claws-Mail offers almost all the features one might expect from an email client such as IMAP4/POP3 support, filters, custom templates and so on. Furthermore, the application comes with a plugin system allowing the user to add extra functionalities such as a calendar, RSS aggregator. You can view the full list of plugins by visiting this page.

To install Claws-Mail on Ubuntu, simply run the following commands in a terminal:

sudo add-apt-repository ppa:claws-mail/ppa
sudo apt-get update 
sudo apt-get install claws-mail
