Are you looking for a simple alarm clock app? If so, then Alarm Clock is for you. It allows you to set custom alarms so that you will never miss an appointment.

Alarm Clock is an easy to use utility with a clean interface that lets you easily create and manage a list of alarms.

You can choose between two types of alarms: Clock or Timer. The Clock is just like any other normal alarm clock that will activate at a specific time while the Timer is simply a countdown alarm.

Alarm Clock is available in the Ubuntu Software Center (search for alarm-clock-applet). To install it from the command-line, type the following command into a terminal window:


sudo apt-get install alarm-clock-applet
