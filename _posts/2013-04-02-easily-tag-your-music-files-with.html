Picard is the official music tagger of MusicBrainz which is a community-driven database of music information (i.e metadata). 
Picard supports almost all known audio formats. It pulls the metadata from MusicBrainz allowing the user to use AcoustID (i.e. audio fingerprints) and thus making the the entire tagging process efficient, and quick.


Picard's Key Features:

  • Plugin system
  • Drag & Drop support
  • Preview mode 
  • Full Unicode support
  • Ability to submit discID
  • Translation

To install Picard on Ubuntu, simply add the following PPA to your Software Sources:

sudo add-apt-repository ppa:musicbrainz-developers/stable
sudo apt-get update
sudo apt-get install picard
