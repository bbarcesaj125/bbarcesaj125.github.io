Turpial is featureful microblogging client written in Python. It comes with a beautiful GTK+ interface that allows you to interact with you Twitter/Identica account in many ways such as updating your status, uploading new photos, following a new user ...

Turpial includes the following features:

  • Huge list of URL shortener services
  • Customizable interface (e.g. One column view, wide view ...)
  • Desktop notifications (e.g. Bubble notifications, sound notifications)
  • Tweet filter (i.e. Temporarly block tweets from a given user)
