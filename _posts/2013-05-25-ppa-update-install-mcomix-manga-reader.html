This is a quick update for those of you who have used our PPA to install Mcomix. Mcomix 1.0 has been released a few weeks ago and so I have uploaded its latest version to the TooPPA which provides binaries for Ubuntu Precise, Quantal, Raring and Saucy.

Mcomix is a user friendly comic reader that was forked from Comix. It was particularly designed to handle comic/manga files (i.e. Zip, Rar, Cbr ...).

For an overview of Mcomix, you can visit our previous post which includes a quick review of its key features: Mcomix: A Simple And User Friendly Comic/Manga Reader [PPA].

Installation:

To install Mcomix from the TooPPA, simply run the following commands in a terminal:

sudo add-apt-repository ppa:beebarss-saj-ma/tooppa
sudo apt-get update
sudo apt-get install mcomix
