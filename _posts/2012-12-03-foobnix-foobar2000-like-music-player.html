Foobnix is a beautiful audio player for Ubuntu/Linux which has a user interface similar to Foobar2000. It doesn't only play audio files but movies as well. 

Here is a list of the main Foobnix functionalities:
  • Equalizer 
  • Built-in audio converter (from/to: ogg, mp3, mp2, ac3, m4a, wav)
  • Shortcuts support
  • Online music services integration (VKontakte & Last.fm)
  • Tabbed interface
  • Ability to use proxy for all online services
If you want to see the complete changelog of the latest version, please visit this page:
