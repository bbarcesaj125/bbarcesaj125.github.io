Zim is a note taking application which helps you keep a collection of linked notes. Unlike other note taking applications, Zim uses the wiki style to maintain a group of notes. Each note or page can contain links to other pages and you can create an unlimited number of subpages under the same page.

Zim include simple formatting tools like :
  • Headings
  • Bullet lists
  • Checkbox lists
Zim stores all your notes in txt files with the wiki style. Although Zim is a basic application with limited features, it could be used for various tasks such as :
  • Keeping tracks of your movie list
  • Brainstorming
  • Creating To Do lists
  • The possibilities are endless ...
TIP: You can use Zim as a GTD organizer. To do that, you need to create a page for each group of tasks. For example, you can create a page for your 'waiting for' list, another page for your active projects and so on. For more information, i suggest you read this article which is a very detailed guide on how to implement the GTD method through Zim.
Implementing GTD through Zim

Zim is available on Linux, Windows and Mac. To install it on Ubuntu, add the following PPA to your Software Sources:

sudo apt-add-repository ppa:jaap.karssenberg/zim 
sudo apt-get update 
sudo apt-get install zim
