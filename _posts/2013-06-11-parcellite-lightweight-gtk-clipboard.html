Parcellite is a very simple clipboard manager. It comes with an absolute bare minimum set of features, thus making it a lightweight application with a focus on simplicity and productivity.

Features:

  • Unity's Appindicator support
  • Global shortcuts
  • Ability to restore clipboard content after closing the application (i.e. daemon mode)
  • Clipboard history
  • Custom commands

As you can see in the image above, Parcellite comes with quite a few options such as choosing the clipboard size, saving the history, changing key bindings and so on.

To install Parcellite on Ubuntu, simply run the following command in a terminal:

sudo apt-get install parcellite

If you want to automatically get the latest version as soon as it's available, you can add the following ppa to your software sources: 

sudo add-apt-repository ppa:rickyrockrat/parcellite-appindicator
sudo apt-get update
sudo apt-get install parcellite
